# CheatSheet Shortcut Keyboard in IDE Xcode

CheatSheet shortcut keyboard in IDE XCode. Happy Type :grin:

### Hide/Show Debug Console

Keybinding

``` bash
cmd + shift + y
```

### Hide/Show Project Navigator

Keybinding

``` bash
cmd + 0
```

### Hide/Show Utilities

Keybinding

``` bash
cmd + opt + 0
```

### Auto Indent (Re-Indent)

Keybinding

``` bash
cmd + i
```

### File Open Quickly 

Keybinding

``` bash
cmd + shift + o
```

### Back Open File 

Keybinding

``` bash
cmd + ctrl + arrow left
```

### Forward Open File 

Keybinding

``` bash
cmd + ctrl + arrow right
```
